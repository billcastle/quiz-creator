# QZ-0005 — Create core agent definitions (architect, ai-dev, ai-qa)

## Description

Create or update the three core agent definition files in `.claude/agents/`. These are the highest-stakes definitions in the project: the Architect governs all technical decisions, ai-dev writes all application code, and ai-qa validates all output. All three must be Questify-specific — general-purpose agent prompts are not acceptable.

The `architect.md` file currently contains a generic system architecture reviewer prompt and must be fully rewritten to match the PHASE-02 spec. The `ai-dev.md` and `ai-qa.md` files must be reviewed against the PHASE-02 spec and updated if they are missing any required sections.

Reference: `docs/architecture/PHASE-02-agent-configuration.md` — Technical Architecture → Agent Definitions.

## Acceptance Criteria

- [ ] `.claude/agents/architect.md` has `model: claude-opus-4-8`, Questify-specific persona, full responsibilities list, project context (monorepo, Cloudflare, QZ tickets), and constraints (never modifies source code, never creates tickets directly, never overrides an Accepted ADR without a superseding ADR)
- [ ] `.claude/agents/ai-dev.md` has `model: claude-sonnet-4-6`, Nova/Sage/Milo personas, monorepo cardinal placement rule, all 7 skills listed, QZ branch/commit/PR conventions, and all constraints
- [ ] `.claude/agents/ai-qa.md` has `model: claude-sonnet-4-6`, Ivy persona, bug report format, Playwright test naming (`apps/web/tests/`), sign-off comment format, and constraints (never modifies source code, never approves a phase with open critical/high bugs)
- [ ] All three files pass the PHASE-02 spec review by Architect
- [ ] No placeholder text left in any file

## Out of Scope

- `ai-ba.md`, `ai-devops.md`, `ai-readme.md` — covered in QZ-0006
- Creating the four missing skills — covered in QZ-0007
- Any application code in `apps/` or `packages/`

Files already exist from the foundation commit. This ticket's job is to verify and update them to meet the PHASE-02 spec. Architect.md in particular needs a full rewrite.
