# QZ-0014 — Create missing skills: drizzle-orm, cloudflare-workers, shadcn-ui, biome

## Description

Four skills identified in the PHASE-02 skills audit are not yet installed in `.claude/skills/` or `.agents/skills/`. These skills provide agents with curated, project-specific guidance for technologies they will use in implementation phases.

Each skill file should cover: setup/installation, configuration patterns specific to this project, common patterns and anti-patterns, and project-specific conventions that the agent would otherwise need to discover by trial and error.

This ticket is **not blocking PHASE-02 or PHASE-03 completion**, but `biome` and `drizzle-orm` should be completed before implementation begins in PHASE-03 and PHASE-06 respectively.
